# 🍽️ The Lillique
A basic frontend restaurant website built using HTML, CSS, and JavaScript. The project focuses on strengthening core frontend development skills while exploring how real-world restaurant websites are structured and designed.

## 🛠️ Technologies Used
### Frontend
* HTML5
* CSS3
* Vanilla JavaScript ( No frameworks)

### Development Tools
* GitHub
* VS Code

## 🎨 Theme
A warm and cozy restaurant-inspired design featuring soft earthy tones, clean layouts, and subtle interactions to create an inviting dining experience.

## ✨ Key Features
* Fully responsive design for all screen sizes
* Interactive navigation bar with smooth scrolling
* Dynamic food menu section with organized categories
* JavaScript-powered cart functionality
* Add and remove items from cart
* Hover effects and smooth UI transitions
* Multi-section layout including Home, Menu, About, Events, and Contact sections
* Clean and structured layout built using Flexbox and Grid

## 🧠 What I Learned
Through this project, I gained practical experience in:
* Structuring a complete multi-section website from scratch
* DOM manipulation using JavaScript
* Handling user interactions and events
* Creating responsive layouts using CSS
* Applying basic UI/UX principles for better user experience
* Writing cleaner, reusable, and maintainable frontend code

## 🚧 Challenges Faced
* Designing layouts that remain responsive across different screen sizes
* Implementing cart functionality using JavaScript
* Organizing CSS efficiently as the project expanded
* Maintaining consistency throughout the website's design and user experience
